1. In properties of module "TS3 Dream Launcher", change the version string to the new version like "1.X.X"
2. Compile "Dl3DxOverlay", "Sims3Launcher" and "TS3 Dream Launcher" modules in RELEASE mode
3. Get "release" resultant files of each module and copy all files to folder "Dream-Launcher-Compiled" in this repository
4. Zip all files of the folder "Dream-Launcher-Compiled"
5. Copy the Zip file to folder "Repository-Pages" with the name of "launcher-current-compilation.zip"
6. Update the file "launcher-current-version.txt" to have the current version string of current version.
